## 版本配套关系说明
JMesh目前支持 Spring Cloud 最新版本。Spring Cloud 、Spring Boot 及 JMesh SDK 版本之间的关系如下表所示。

| Spring Cloud | Spring Boot |
| ------------ | ----------- |
| 2020.0.x     | 2.5.x       |

| SDK 版本号 | 新增特性                                                     |
| ---------- | ------------------------------------------------------------ |
| 1.0.0      | 支持基于Consul的服务注册与发现，服务限流，熔断，服务路由，服务鉴权等功能 |
| 1.1.0      | 融合MeshWare服务治理SDK                                      |
| 1.2.0      | 加入Nacos注册中心支持模块                                    |
